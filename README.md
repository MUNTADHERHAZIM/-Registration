# 🎓 نظام إدارة شؤون الطلاب الجامعي
## University Student Affairs Management System

نظام متكامل ومحترف لإدارة شؤون الطلاب وعمليات التسجيل الجامعي، مبني بإطار عمل Django مع تصميم عصري وكامل الميزات.

---

## ✨ المميزات الرئيسية

### 📊 لوحة تحكم تفاعلية
- إحصاءات شاملة في الوقت الفعلي
- رسوم بيانية تفاعلية (توزيع الطلاب، الجنس، المراحل، الإقبال الشهري)
- عرض آخر الطلاب المسجلين

### 👥 إدارة الطلاب
- إضافة / تعديل / حذف بيانات الطلاب الكاملة
- ملف شخصي مفصّل لكل طالب مع صورة
- البحث والفلترة المتقدمة (بالاسم، الرقم، القسم، المرحلة، الحالة، الجنس)
- إدارة حالة الطالب (نشط / موقوف / متخرج / منسحب)
- رفع وإدارة الوثائق الطلابية
- التصدير إلى PDF و Excel

### 🏛️ إدارة الأقسام
- إضافة وتعديل الأقسام الأكاديمية
- عرض إحصاءات كل قسم (الطلاب، المواد، المدرسون)

### 📚 إدارة المواد الدراسية
- قاعدة بيانات كاملة للمواد مع المتطلبات السابقة
- إدارة الشعب الدراسية (section) مع القاعات والجداول
- عرض حالة الطاقة الاستيعابية لكل شعبة

### 👨‍🏫 الكادر التدريسي
- ملفات شخصية للمدرسين مع الألقاب العلمية
- ربط المدرسين بالشعب والمواد

### 📋 التسجيل
- تسجيل الطلاب في المواد مع التحقق من الطاقة الاستيعابية
- إدارة حالة التسجيل (مسجل / مكتمل / راسب / محذوف)
- عرض تسجيلات الطالب لكل الفصول

### ⭐ الدرجات
- إدخال الدرجات بشكل جدولي سريع مع حساب فوري للمجموع والتقدير
- سلم تقدير احترافي (A+ إلى F)
- نشر الدرجات مع تحديث حالة التسجيل تلقائياً
- كشف درجات تفصيلي لكل طالب بالفصول
- حساب المعدل التراكمي GPA تلقائياً

### 📊 التقارير
- تقرير الطلاب مع تصدير PDF و Excel
- الإحصاءات العامة للأقسام والمراحل
- تقارير الدرجات مع نسب النجاح والرسوب
- أفضل الطلاب (أعلى المعدلات)

### 👤 إدارة المستخدمين
- نظام صلاحيات متعدد المستويات (مدير / موظف / مستخدم)
- إنشاء وإدارة المستخدمين
- الملف الشخصي وتغيير كلمة المرور

## 🚀 طريقة التشغيل

### الطريقة السريعة (Linux/Mac):
```bash
chmod +x run.sh
./run.sh
```

### الطريقة اليدوية:
```bash
# 1. تثبيت المكتبات
pip install -r requirements.txt

# 2. إنشاء قاعدة البيانات
python manage.py makemigrations
python manage.py migrate

# 3. توليد البيانات التجريبية
python seed_data.py

# 4. تشغيل الخادم
python manage.py runserver
```

## 🛠️ صيانة النظام وإعادة ضبط قاعدة البيانات

في حالة الرغبة في مسح كافة البيانات والبدء من جديد (تصفير النظام)، اتبع الخطوات التالية:

### 1️⃣ إعادة ضبط كاملة (حذف كل شيء)
إذا أردت مسح جميع المستخدمين، الإعدادات، والطلاب:
1. قم بحذف ملف `db.sqlite3`.
2. نفذ الأمر التالي لإنشاء الجداول من جديد:
   ```bash
   python manage.py migrate
   ```
3. أنشئ حساب مدير جديد:
   ```bash
   python manage.py createsuperuser
   ```

### 2️⃣ مسح الطلاب والوثائق فقط (مع تصفير العدادات)
إذا أردت مسح بيانات الطلاب فقط والاحتفاظ بالإعدادات والأقسام، نفذ هذا الأمر في شاشة الأوامر:
```bash
python manage.py shell -c "from students.models import Student, StudentDocument; StudentDocument.objects.all().delete(); Student.objects.all().delete(); from django.db import connection; cursor = connection.cursor(); cursor.execute('DELETE FROM sqlite_sequence WHERE name=\"students_student\"'); cursor.execute('DELETE FROM sqlite_sequence WHERE name=\"students_studentdocument\"'); print('تم مسح بيانات الطلاب بنجاح')"
```

### 📋 الترتيب الصحيح لإدخال البيانات
لتجنب أخطاء الربط (Foreign Keys)، يجب إدخال البيانات بالترتيب التالي:
1. **الأقسام الأكاديمية**: (هندسة، طب، الخ).
2. **السنوات الدراسية**: (2024-2025، الخ).
3. **الفصول الدراسية**: (الفصل الأول، الثاني).
4. **الطلاب**: (بعد التأكد من وجود القسم والسنة المناسبة).
